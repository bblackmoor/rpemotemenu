# RP Emote Menu

**RP Emote Menu addon for World Of Warcraft (WOW)**

The command to toggle display of the RP Emote Menu is 
```
/emotes
```
My next steps will be:

- making it so that the categories and emotes can be edited by the user
- an optional minimap button to toggle showing the menu
- a setting for fonts and/or font sizes
